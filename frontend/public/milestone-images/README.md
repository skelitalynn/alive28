# 里程碑 NFT 图片资源

请在此目录下放置 10 张里程碑 NFT 图片，文件命名格式：

- `milestone-1.png`
- `milestone-2.png`
- `milestone-3.png`
- `milestone-4.png`
- `milestone-5.png`
- `milestone-6.png`
- `milestone-7.png`
- `milestone-8.png`
- `milestone-9.png`
- `milestone-10.png`

## 图片要求

- 格式：PNG（推荐）或 JPG
- 尺寸：建议 512x512 或 1024x1024
- 风格：温馨、治愈、符合 Alive28 品牌调性
- 内容：可以包含里程碑相关的图标、文字或图案

## 使用方式

系统会在铸造里程碑 NFT 时随机选择一张图片。

如果图片不存在，系统会自动显示 SVG 占位符。
